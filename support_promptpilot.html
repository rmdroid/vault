Häufige Fragen

Wie funktioniert PromptPilot?

PromptPilot nutzt Gemini Nano – Googles lokal laufende KI in Chrome – um Prompts direkt im Browser zu verbessern, strukturieren und zu erweitern. Alle Verarbeitungsschritte finden auf Ihrem Gerät statt, ohne externe Server.

Warum funktioniert PromptPilot nur in Chrome?

Gemini Nano ist eine Chrome-exklusive Funktion und steht aktuell nur in Google Chrome ab Version 127 zur Verfügung. Andere Browser wie Brave, Firefox oder Safari unterstützen diese API nicht.

Was sind die Hardwareanforderungen?

Gemini Nano benötigt mindestens 16 GB RAM und 4 GB VRAM. Beim ersten Aktivieren des Plugins lädt Chrome das KI-Modell herunter – das kann einige Minuten dauern.

Werden meine Texte an Server übertragen?

Nein. Die gesamte KI-Verarbeitung läuft lokal über Gemini Nano auf Ihrem Gerät. Es werden keine Prompt-Inhalte an externe Server gesendet. PromptPilot überträgt keine Nutzerdaten.

Kann ich PromptPilot auf dem iPad oder mobil nutzen?

Nein. Chrome Extensions sind auf mobilen Geräten – einschließlich iPads – nicht verfügbar. PromptPilot ist ausschließlich für Chrome auf dem Desktop (Windows, Mac, Linux) konzipiert.

Hinweise

  • PromptPilot erfordert Google Chrome – andere Browser werden nicht unterstützt.
  • Die KI-gestützte Prompt-Optimierung ersetzt keine inhaltliche, rechtliche oder datenschutzrechtliche Prüfung der erzeugten Texte.
  • Die Qualität der Ausgabe hängt von der installierten Gemini-Nano-Version und den verfügbaren Hardware-Ressourcen ab.

Datenschutz auf einen Blick

  • Die gesamte KI-Verarbeitung erfolgt lokal über Gemini Nano – keine Cloud, keine externen Server.
  • Es gibt kein Tracking, keine Analyse-Software und keine Werbung.
  • Es gibt keine Registrierung, kein Nutzerkonto und kein Abo.
  • Prompt-Inhalte werden nicht gespeichert oder übertragen.

Datenschutzerklärung

Diese Seite dient zugleich als Support-Seite und als Datenschutzerklärung für PromptPilot. Die konkreten Browser-Berechtigungen ergeben sich ergänzend aus dem Chrome Web Store.

Welche Daten die Erweiterung verarbeitet

PromptPilot verarbeitet ausschließlich Text, den Nutzerinnen und Nutzer aktiv in die Erweiterung eingeben oder der in einem aktiven Tab markiert ist. Die Verarbeitung erfolgt lokal über die Gemini-Nano-API von Chrome.

Wie diese Daten verarbeitet werden

Die eingegebenen Texte werden an die lokale LanguageModel-API von Chrome übergeben und direkt auf dem Gerät verarbeitet. Eine Übertragung an externe Server findet nicht statt. Es werden keine Inhalte dauerhaft gespeichert.

Übermittlung an Dritte

Es erfolgt keine Übertragung von Nutzerdaten oder Prompt-Inhalten an eigene Server oder Dritte. PromptPilot stellt keine Verbindung zu externen Diensten her.

Website-Inhalte und aktive Tabs

Die Erweiterung kann auf ausdrückliche Nutzeraktion markierten Text aus einer Webseite übernehmen. Es gibt keinen automatischen Zugriff auf Browserverlauf, keine Hintergrundanalyse von Webseiten und kein Nutzungsprofiling.

Support-Anfragen

Wenn Sie per E-Mail oder Kontaktformular eine Support-Anfrage senden, werden die von Ihnen übermittelten Angaben ausschließlich zur Bearbeitung Ihres Anliegens verwendet.

Cookies, Tracking und Werbung

Die Erweiterung selbst verwendet kein Tracking, keine Analyse-Software und keine Werbung. Diese Support-Seite verwendet keine Cookies und keine Nutzerkonten.
