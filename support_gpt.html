Häufige Fragen

Wie arbeitet GPT Vault?

GPT Vault speichert Skills, Prompts, Kategorien und Tags lokal im Browser. Inhalte können gesucht, strukturiert, exportiert und bei Bedarf direkt in eine geöffnete KI-Oberfläche eingefügt werden.

Werden meine Inhalte an Server übertragen?

Nein. Die Erweiterung speichert ihre Daten lokal in chrome.storage.local. Externe Abrufe erfolgen nur dann, wenn Sie selbst eine Team- oder Paid-URL hinterlegen. Dabei werden keine Inhalte an eigene Server übertragen.

Kann GPT Vault Inhalte aus Webseiten lesen?

Nur auf ausdrückliche Nutzeraktion. Über das Kontextmenü kann markierter Text übernommen werden. Es gibt keine automatische Analyse des Surfverhaltens, kein Tracking und keinen Zugriff auf den Browserverlauf.

Ist ein eingefügter Prompt sofort korrekt oder freigegeben?

Nein. GPT Vault ist ein Organisations- und Einfügewerkzeug. Die fachliche, rechtliche und datenschutzbezogene Prüfung der Inhalte bleibt immer beim Nutzer.

Hinweise

  • GPT Vault ersetzt keine fachliche, rechtliche oder datenschutzrechtliche Prüfung.
  • Gespeicherte Skills und Prompts sollten vor produktiver Nutzung immer überprüft und an den konkreten Kontext angepasst werden.
  • Team- oder Premium-Listen sollten nur aus vertrauenswürdigen Quellen eingebunden werden.

Datenschutz auf einen Blick

  • Die Erweiterung speichert Skills, Prompts, Kategorien, Tags sowie optionale URLs lokal im Browser.
  • Es gibt kein Tracking, keine Analyse-Software und keine Werbung.
  • Es gibt keine Registrierung, kein Nutzerkonto und kein Abo innerhalb der Erweiterung.
  • Inhalte werden nicht an eigene Server übertragen, um die Kernfunktion der Erweiterung auszuführen.
  • Externe Inhalte werden nur geladen, wenn der Nutzer selbst eine URL für Team- oder Premium-Inhalte hinterlegt.

Datenschutzerklärung

Diese Seite dient zugleich als Support-Seite und als Datenschutzerklärung für die Chrome-Erweiterung GPT Vault.

Welche Daten die Erweiterung verarbeitet

GPT Vault verarbeitet Inhalte, die Nutzerinnen und Nutzer selbst eingeben, speichern, importieren, markieren oder in eine geöffnete KI-Oberfläche einfügen. Dazu gehören insbesondere Skills, Prompts, Kategorien, Tags, interne Notizen und optional hinterlegte Team- oder Premium-URLs.

Wie diese Daten verarbeitet werden

Diese Inhalte werden lokal im Browser in chrome.storage.local gespeichert. Die Erweiterung verwendet keine eigenen Server zur Speicherung Ihrer Inhalte. Wenn Sie die Exportfunktion nutzen, werden die von Ihnen gespeicherten Inhalte lokal als JSON- oder TXT-Datei bereitgestellt.

Übermittlung an Dritte

Es erfolgt keine Übertragung Ihrer Inhalte an eigene Server. Externe Abrufe erfolgen nur dann, wenn Sie selbst eine Team- oder Premium-URL hinterlegen. In diesem Fall ruft die Erweiterung die von Ihnen angegebene JSON-Quelle direkt ab, um dort enthaltene Skill-Inhalte zu laden. Eine weitergehende Weitergabe, ein Verkauf oder eine Nutzung zu Werbe- oder Trackingzwecken findet nicht statt.

Website-Inhalte und aktive Tabs

Die Erweiterung kann auf ausdrückliche Nutzeraktion markierten Text aus einer Webseite übernehmen oder gespeicherte Inhalte in die aktuell geöffnete Chat-Oberfläche einfügen. Es gibt keinen automatischen Zugriff auf Browserverlauf, keine Hintergrundanalyse von Webseiten und kein Nutzungsprofiling.

Support-Anfragen

Wenn Sie per E-Mail oder Kontaktformular eine Support-Anfrage senden, werden die von Ihnen übermittelten Angaben ausschließlich zur Bearbeitung Ihres Anliegens verwendet.

Cookies, Tracking und Werbung

Die Erweiterung selbst verwendet kein Tracking, keine Analyse-Software und keine Werbung. Diese Support-Seite selbst verwendet keine Cookies und keine Nutzerkonten. Bei Nutzung externer Kontaktwege gelten ergänzend die Datenschutzbestimmungen der jeweiligen Anbieter.
