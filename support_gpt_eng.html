Frequently asked questions

How does GPT Vault work?

GPT Vault stores skills, prompts, categories, and tags locally in the browser. Users can search, structure, export, and insert saved content directly into an open AI interface when needed.

Are my contents sent to servers?

No. The extension stores its data in chrome.storage.local. External requests only happen if you deliberately add a team or paid URL yourself. No content is sent to the developer’s own servers.

Can GPT Vault read website content?

Only on explicit user action. The context menu can import selected text. There is no automatic browsing analysis, no tracking, and no access to browser history.

Does a saved prompt mean the output is correct or approved?

No. GPT Vault is an organization and insertion tool. Professional, legal, and privacy review always remains with the user.

Important notes

  • GPT Vault does not replace professional, legal, or privacy review.
  • Saved skills and prompts should always be checked and adapted to the real use case before production use.
  • Team or premium content lists should only be connected from trusted sources.

Privacy at a glance

  • The extension stores skills, prompts, categories, tags, and optional URLs locally in the browser.
  • There is no tracking, analytics software, or advertising.
  • There is no registration, no user account, and no subscription inside the extension.
  • Content is not transmitted to the developer’s own servers in order to provide the core functionality.
  • External content is only loaded when the user explicitly adds a URL for team or premium content.

Privacy Policy

This page serves both as the support page and the privacy policy for the GPT Vault Chrome extension.

What data the extension processes

GPT Vault processes content that users explicitly enter, save, import, select, or insert into an open AI interface. This includes skills, prompts, categories, tags, internal notes, and optional team or premium URLs.

How this data is processed

This content is stored locally in the browser using chrome.storage.local. The extension does not use its own servers to store your content. If you use the export feature, your stored content is made available locally as a JSON or TXT file.

Transfer to third parties

Your content is not transmitted to the developer’s own servers. External requests only occur if you deliberately add a team or premium URL. In that case, the extension directly fetches the JSON source you specified in order to load the included skill content. There is no broader sharing, sale, advertising use, or tracking use of your data.

Website content and active tabs

On explicit user action, the extension can import selected text from a webpage or insert stored content into the currently open chat interface. There is no automatic access to browsing history, no background analysis of websites, and no profiling of users.

Support requests

If you send a support request by email or through the contact form, the information you provide is used solely to handle your request.

Cookies, tracking, and advertising

The extension itself uses no tracking, no analytics software, and no advertising. This support page itself uses no cookies and no user accounts. If you use external contact channels, the privacy policies of those providers also apply.
